協助主題：看板

說明

看板就是出現在頁面上方的宣傳廣告。他們可以是圖片或Flash，例如預設的三則看板就有一個是Flash的看板。XOOPS有內建一個簡單的看板管理系統，讓您可以建立託播者（或廣告主）的資料。您可以同時放上多個看板，只要使用者連到網站上，就會隨機秀出其中一則看板，每連到一個新頁面，就會隨機載入看板內容。

當看板每呈現一次，就會在「展示」計數一次，這可以讓您或託播者清楚的知道此看板呈現了多少次。一般來說，若有人託播，那麼都是以「展示」次數來計算費用的。

請注意：假如您是打算用在商業上（例如販售廣告業務），那麼有個重要設定您應該去設一下：「後台→偏好設定→系統設定→一般設定→您的 IP 位址」您應該填入您網站的 IP 位置，以避免自己在修改或瀏覽網站時算到展示次數中。若沒有這樣做，將會被認為是不老實的！

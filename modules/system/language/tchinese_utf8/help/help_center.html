如何開始使用 XOOPS?

XOOPS 是一個用PHP搭配MySQL資料庫所寫的網頁平台，它是以物件導向方式來開發，所以是一個可以建立小至個人網站大至大型社群網站、公司網站或入口網站的理想工具。為了更有效率的應用XOOPS，您需要安裝一些模組以提供網站所需要的功能。在官網的 Module Repository 有許多不同的模組，例如行事曆、相簿、新聞、公佈欄、天氣...等各式各樣的模組。

如何使用XOOPS協助中心？

此協助中心是提供各個管理功能的基本資訊，不管什麼時候您想了解目前您所在的功能頁面到底有什麼功能，您只要點擊右上角的圖示即可開啟此說明。r

如何在XOOPS中建立內容？

您需要去下載並安裝相關的模組才行！例如新聞模組、或內容模組...等。
